This is a recipe sharing website built using MERN stack technology.

Current Website:
 Functional Registration Page.
 Functional Login Page.
 Main Home Page where you can add recipes.
 Recipes categorized as vegetarian and non-vegetarian.
 Functional search bar for fast search for a particular recipe.

Future updations:
 AI integrated recipe suggestion.
 Calorie tracker for each recipe
 Chef user interaction through chatbox to clear doubts regarding recipes or to provide feedback to the Recipe maker.

How to run the code:
 Pull both Frontend and Backend code to the computer.
 Save both the files in a directory.
 Start the frontend using 'npm start'.
 Start the backend using 'nodemon server.js'.
